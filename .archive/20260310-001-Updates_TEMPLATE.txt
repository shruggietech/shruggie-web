CODING CONVENTIONS
==================
- All files: UTF-8 encoding without BOM. LF line endings.
- TypeScript: Strict mode. No `any` types. Use `satisfies` for type narrowing where appropriate.
- Imports: React/Next.js first, then third-party libraries, then local imports (`@/`).
- Components: Functional components with named exports (default export for page components).
  Use `forwardRef` for components that wrap native HTML elements (Button, Card).
- Tailwind: Use design system tokens from `tailwind.config.ts` (e.g., `text-text-primary`,
  `bg-bg-elevated`, `text-accent`). Do not use raw hex values in className strings.
- Accessibility: Every interactive element must be keyboard-focusable. Every image must have
  alt text (or `alt=""` with `aria-hidden="true"` for decorative images). Every form field
  must have an associated `<label>`. Focus indicators use brand green (#2BCC73).
- Animations: All animations must respect `prefers-reduced-motion`. Use Framer Motion's
  `useReducedMotion` hook or CSS `@media (prefers-reduced-motion: reduce)`.
